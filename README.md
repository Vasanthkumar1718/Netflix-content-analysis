# Netflix Content Strategy Analysis

## Objective
Analyze Netflix content to understand growth trends, content distribution, and strategy over time.

## Dataset
Netflix Movies and TV Shows Dataset (Kaggle)

## Tools Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Movies dominate Netflix content compared to TV shows.
- Content increased significantly after 2014.
- Most content is recent and concentrated in the 2010s.
- United States produces the highest amount of content.

## Project Workflow
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis
- Visualization
- Insights

## Conclusion
Netflix focuses on recent content and shows rapid growth after 2014, with a strong emphasis on movies.
